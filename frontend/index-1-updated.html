What is

Fuzzy Ensembling

Multi-Model Architecture

Our system combines predictions from seven deep learning models (DenseNet-169, InceptionV3, Xception, InceptionResNetV2, VGG19, MobileNetV2, ResNet50V2) to leverage their strengths.

Fuzzy Logic Integration

We use fuzzy logic rules to weigh each model's output based on its performance with similar images, enhancing prediction accuracy.

Uncertainty Quantification

Our fuzzy ensemble provides well-calibrated confidence scores, reflecting prediction uncertainty accurately.

Classification Process

1. Image Upload

Upload a microscopic image of cervical cells in JPG, PNG, or BMP format.

2. Image Preprocessing

The image is resized to 224x224 pixels and normalized for CNN input.

3. Model Prediction

Seven CNN models analyze the image, generating prediction probabilities for each cell class.

4. Fuzzy Ensembling

Model predictions are combined using fuzzy membership functions for enhanced accuracy.

5. Results & Similar Images

View the classification result and similar images for medical analysis.
